# Activities Folder

Place activity/event photos here:

## Naming Convention:
- Use descriptive names with date: `tournament-2025-winter.jpg`
- Or event type: `workshop-game-dev-nov2025.jpg`

## File Specifications:
- Format: JPG or PNG
- Size: 1200x800px to 1920x1080px (landscape)
- Max file size: 500KB

## Usage:
```tsx
<Image src="/activities/tournament-2025-winter.jpg" alt="Winter Tournament" width={600} height={400} />
```

## Example Files:
- tournament-2025-winter.jpg
- workshop-game-dev.jpg
- seminar-career-opportunities.jpg
- charity-gaming-marathon.jpg
