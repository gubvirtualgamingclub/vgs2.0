# Contributors Profile Images

## Instructions

Add profile photos for the contributors in this folder with the following names:

1. **tomal.jpg** - Faysal Hossain Tomal's profile photo
2. **sazib.jpg** - MD. Sazib's profile photo

## Image Requirements

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 400x400 pixels (square)
- **Aspect Ratio**: 1:1 (square)
- **File Size**: Keep under 500KB for optimal performance
- **Quality**: High quality, professional photo

## Example

```
/public/contributors/
├── tomal.jpg  (Faysal Hossain Tomal)
├── sazib.jpg  (MD. Sazib)
└── README.md  (this file)
```

## Note

If no images are provided, the page will display placeholder avatars with the contributors' initials (FH for Faysal, MS for MD. Sazib).
