# Members Folder

Place committee member photos here:

## Naming Convention:
- Use descriptive names: `president-2025.jpg`, `vice-president-2025.jpg`
- Or use member names: `john-doe.jpg`, `jane-smith.jpg`

## File Specifications:
- Format: JPG or PNG
- Size: 400x400px to 800x800px (square)
- Max file size: 200KB

## Usage:
```tsx
<Image src="/members/president-2025.jpg" alt="President" width={200} height={200} />
```

## Example Files:
- president-2025.jpg
- vice-president-2025.jpg
- secretary-2025.jpg
- treasurer-2025.jpg
