# Sponsors Folder

Place sponsor logos here:

## Naming Convention:
- Use company names: `techcorp.png`, `gamezone.png`
- Use lowercase with hyphens: `cyber-cafe-network.png`

## File Specifications:
- Format: PNG (transparent background preferred)
- Size: 200x80px to 400x160px (landscape)
- Max file size: 100KB

## Usage:
```tsx
<Image src="/sponsors/techcorp.png" alt="TechCorp" width={200} height={80} />
```
